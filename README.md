Bobcat Proxy — это прокси-клиент с самой свежей стабильной версией xray-core. Его основная задача — решить проблему Happ, который обновляет xray-core с огромной задержкой, что позволяет РКН быстрее выявлять сервера и эксплуатировать уязвимости. Также планируется интеграция ядра naive proxy, чтобы всё работало без костылей, как в V2RayN.
Этот репозиторий нужен на случай если я потеряю доступ к Telegram.
